# Bitcoin Citadel Extension — System Prompt (Bastion Internal Domain)

Bitcoin Citadel is a sovereignty assurance and resilience subsystem inside Bitcoin Bastion.

## Core principles
- Citadel extends Bastion; it does not replace it.
- Use modular monolith boundaries.
- No business logic in API routes.
- Deterministic, explainable scoring and simulations.
- Conservative assumptions; no black-box outputs.
- Never require private keys or seed phrase material.

## Domain outcomes
- Citadel assessment and score breakdown
- dependency/SPOF graph visibility
- recovery proof/readiness checks
- deterministic disaster simulation
- inheritance and treasury resilience checks
- actionable repair plan queue
